## Exercise

An agent with strictly concave utility $u$ and wealth $w$ faces a loss $L$ with
probability $\pi \in (0,1)$. She can buy coverage $q \in [0, L]$ at premium
$\gamma q$, so her wealth is $w - \gamma q - L + q$ if the loss occurs and
$w - \gamma q$ otherwise.

1. Write the agent's expected utility as a function of $q$ and derive the
   first-order condition.
2. Show that if the premium is actuarially fair ($\gamma = \pi$), the agent chooses
   full coverage, $q^* = L$.
3. Show that if $\gamma > \pi$, any solution has $q^* < L$. Interpret.
4. Where exactly does the argument use strict concavity of $u$? Exhibit a
   risk-neutral counterexample to part 2's conclusion of *unique* optimality.
